
# debezium-mongodb-example

It is a project that aims to learn how to use Debezium and mongo db.




## Installation

For running whole app layer

```bash
  docker-compose up
```

## Run Locally

Need to send POST with json

```bash
  http://localhost:8083/connectors/
```
